python虚拟环境

发表于 2019-02-07

一、为什么要用python虚拟环境

在实际项目开发中，我们通常会根据自己的需求去下载各种相应的框架库，如Scrapy、Beautiful Soup等，但是可能每个项目使用的框架库并不一样，或使用框架的版本不一样，或者使用的python版本不同，这样需要我们根据需求不断的更新或卸载相应的库。
直接对我们的Python环境操作会让我们的开发环境和项目造成很多不必要的麻烦，管理也相当混乱。

二、虚拟环境的安装

  1. 提前安装好python环境，并把路径加入到环境变量;
  2. 安装virtualenv
    pip install virtualenv
  3. 创建虚拟环境
    virtualenv 虚拟环境名这个命令创建虚拟环境，会在当前目录创建
  4. 先要进入到虚拟环境目录的Scripts中，然后在激活虚拟环境activate.bat,则进入到新建的虚拟环境中，之后就可以通过pip命令安装各种依赖包。
  5. 退出虚拟环境deactivate.bat.

三、虚拟环境的配置

  1. 安装 virtualenvwrapper-win
    pip install virtualenvwrapper-win可以使用国内镜像下载pip install -i 网址 virtualenvwrapper-win
  2. 使用workon查看是否安装成功
  3. 通过新方法创建虚拟环境mkvirtualenv 虚拟环境名
  4. 为虚拟环境配置一个专门的目录
    • 新建一个要存放的文件夹
    • 配置电脑系统环境变量
      变量名：WORKON_HOME
      变量值：目录地址
      再次运行WORKON，发现目录下没有文件夹，可以把之前的虚拟环境复制过来
  5. 进入某个虚拟空间
    workon 虚拟环境名
  6. 退出虚拟空间deactivate
